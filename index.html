Comandos Básicos


  • < HTML > ... < /HTML > : Indica el comienzo y fin de un archivo HTML
  • < HEAD > ... < /HEAD > : Indica el comienzo y fin de un encabezado (aquí se coloca generalmente el título).
  • < TITLE > ... < /TITLE > : Indica el título, en este caso el título es mi nombre.
  • < BODY > ... < /BODY > : Indica el comienzo y fin del cuerpo de la página, el cuerpo de la página se refiere al contenido de ella, en este caso son los comandos básicos.
  • < P > ... < /P > : Indica comienzo y fin de un párrafo.
  • < BR > : Permite saltarse una línea (se llama quiebre de línea).
  • < Hn > ... < Hn > : Para n entre 1 y 6, hacen que el texto encerrado aparezca como encabezado(un subtítulo).

    • < H1 >...< /H1 >Aquí se muestra un texto con H1

      .

    • < H2 >...< /H2 >Aquí se muestra un texto con H2

      .

    • < H3 >...< /H3 >Aquí se muestra un texto con H3

      .

    • < H4 >...< /H4 >Aquí se muestra un texto con H4

      .
    • < H5 >...< /H5 >Aquí se muestra un texto con H5
      .
    • < H6 >...< /H6 >Aquí se muestra un texto con H6
      .
  • Tipos de letras:
    • < B > ... < /B > : Negrita.
    • < L > ... < /L > : Cursiva.
    • < BLINK > ... < /BLINK > : Parpadeante.
    • < STRONG > ... < /STRONG > : Enfatizada.
  • < UL > ... < /UL > : Indica comienzo y fin de una lista no ordenada (puntos). Dentro de ellos, cada item empieza por < LI > y termina al terminar la línea.
  • < OL > ... < /OL > : Indica comienzo y fin de una lista ordenada (números). Dentro de ellos, cada item empieza por < LI > y termina al terminar la línea.
    Ejemplo: 
    				  < OL >
					< LI > Primer item
					< LI > Segundo item
					< UL > 
					  < LI > Primer subitem
					  < LI > Segundo subitem
					< /UL >
					< LI > Tercer item
				  < /OL >

    Y esto se ve

    1. Primer item
    2. Segundo item
      • Primer subitem
      • Segundo subitem
    3. Tercer item

La sintaxis para la inclusión de gráficos es la siguiente:

< img src=archivo.{gif,jpg}Nombre del archivo
width= Ancho del archivo(en puntos
height= Altura del archivos (en puntos)
align= Alineación de la imagen y textos (ver más abajo)
border= Ancho del borde de la imagen
>
Por ejemplo:

< img src="download.jpg" >
